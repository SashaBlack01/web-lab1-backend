## Опис завдання
Розробити серверну частину Web-додатку для платформи публікації оголошень з використанням Django та Django REST Framework. Реалізовано кастомну реєстрацію користувачів, створення оголошень та систему коментарів/лайків.

## Вміст репозиторію
- Python скрипти (Моделі, Відображення, Серіалізатори, Маршрутизатори)
- HTML документація API (Redoc)
- JavaScript скрипт для автоматизації тестування в Postman
- Файл залежностей requirements.txt
